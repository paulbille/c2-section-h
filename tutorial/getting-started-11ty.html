  1. Download and Install NodeJS

  2. Download our 11ty starter-project or start with an empty folder

  3. Open the folder or starter-project in VS Code

  4. Open a terminal in VS Code and:

  5. Run npm init -y to create a package.json file

  1. Run npm install @11ty/eleventy --save-dev to install 11ty

  1. Run npx @11ty/eleventy --serve to start the live-server

If everything goes smooth you should see a blue link to your live-server. You might get an error asking you to install a missing package

Our Starter-Project
...

In our starter project you don't need to worry about all the folders.

You will mainly spend time:

  • configuring your Nunjucks (njk) templates in the _includes/layouts/ folder.
  • writing css or js in public/css/ or public/js/
  • editing your markdown content in content/
  • maybe you will also write some custom code in eleventy.config.js

Attention

This template is set up to work with the frontmatter data that is present in the markdown files in content/blog/ which comes from Jennifer's data that I used for the example. You will have to adjust the Nunjucks template inside _includes/layouts/ to work with your frontmatter data.

To get started on this start looking at:
- post.njk
- postlist.njk

and try to understand what is happening there after familiarizing yourself with some of the material below.

I will dive more into our starter-project on Tuesday but if you want to use 11ty you will have to spend time with it on your own.
