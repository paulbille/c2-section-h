interface
...

noun

  1. a point where two systems, subjects, organizations, etc. meet and interact.

    "the interface between accountancy and the law"

  2. a device or program enabling a user to communicate with a computer.

    COMPUTING

    "a graphical user interface"

    a device or program for connecting two items of hardware or software so that they can be operated jointly or communicate with each other.

    "an application program interface"

note:

  • Definitions of this term extend outside of the browser
  • While most of our concerns will remain here, lets keep the broadest definiton in mind as we proceed

interface
...

verb

  1. interact with (another system, person, organization, etc.).

    "his goal is to get people interfacing with each other"

  2. connect with (another computer or piece of equipment) by an interface.

    COMPUTING

    "the hotel's computer system can interface automatically with the booking system"

note:

  • interacting with the world requires interfacing/an interface
  • so in a sense everything you interact/inteface with has and interface

SketchPad by Ivan Sutherland

note:

  • we start simple with just lines
  • Inspired by As We May Think Sutherland created the first graphical user interface (GUI) for his PhD dissertation.
  • This program is extremely influential across the fields of computation and design. It also marks the first computer aided design (CAD). Above is a link to Alan Kay explaining the design.

The Mother of All Demos

note:

  • The 90-minute presentation essentially demonstrated almost all the fundamental elements of modern personal computing: windows, hypertext, graphics, efficient navigation and command input, video conferencing, the computer mouse, word processing, dynamic file linking, revision control, and a collaborative real time editor (collaborative work).
  • This presentation was the first to publicly demonstrate all of these elements in a single system.

Xerox Alto

note:

  • The Xerox Alto personal workstation is created.
  • This evolving Personal Computer (PC) willemploy the world's first bit-mapped display, graphical user interface with windows and icons, WYSIWYG (What You See is What You Get)
    editor, local area network/file storage, as well as a commercial mouse.

WYSIWYG

note:

  • What You See is What You Get (WYSIWYG) first appear in he Bravo application on the Xerox Alto.
  • It's one of the first major obfuscations distancing the user from understanding how the computer works.
  • Steve Jobs visited Xerox PARC to learn about their interface concepts.
  • He was the most taken with the WYSIWYG and integrated the concept into he design of Macintosh and beyond.
